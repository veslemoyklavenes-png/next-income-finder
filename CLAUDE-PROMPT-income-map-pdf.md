# Task: Add a downloadable "My Income Map" PDF to Next Income Finder

## Context
Next Income Finder is a Next.js app deployed on Vercel. Users answer three questions about their situation, constraints, and skills. The AI generates a ranked income plan with 5 sections: "What I Heard", "Your Ranked Income Plan", "Your Starting Point This Week", "What You Are Working With", and "What to Do Next".

## What to build
After the AI generates the income plan, add a button that lets the user download a clean, one-page PDF summary called "My Income Map." This PDF is a tangible takeaway — something she can print, save, email to herself, or share with a coach.

## PDF content (pull from the generated output)
The PDF should include:

1. **Header**: "My Income Map" + today's date
2. **My Situation** (2-3 sentences): A brief summary pulled from the "What I Heard" section
3. **My Top 3 Options** (ranked list): The top 3 income options from "Your Ranked Income Plan" — just the name, one-line description, and realistic earning range for each
4. **My Constraints** (short list): The key constraints the tool identified, framed positively (e.g. "Works within 3-4 hours/day" not "Can only work 3-4 hours")
5. **My First Step This Week**: The #1 action item from "Your Starting Point This Week"
6. **My Hidden Assets** (2-3 bullets): Key assets from "What You Are Working With"
7. **Footer**: "Generated by Next Income Finder — nextincomefinder.com" + "Update your plan anytime at [URL]"

## Design guidelines
- Clean, warm, readable. Not corporate.
- Use the existing app fonts (Lora for headings, Inter for body)
- Soft colour palette — muted teal or warm earth tones as accent, not bright colours
- Plenty of white space. This should feel calm and clear, not dense.
- One page only. If content is too long, trim descriptions rather than adding a second page.
- The PDF should look good printed on A4 paper

## Technical approach
- Use a client-side PDF library (e.g. jsPDF, or @react-pdf/renderer)
- The download button should appear below the generated output, styled consistently with the app
- Button text: "Download My Income Map (PDF)"
- Generate the PDF from the already-rendered output data — no additional API call needed

## Important
- Do NOT change the existing output format or any other part of the app
- The PDF is an addition, not a replacement
- Make sure the button only appears after the output has been fully generated
- Test that the PDF renders correctly with various content lengths
